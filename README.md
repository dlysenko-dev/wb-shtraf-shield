# WB Штраф-Щит

Telegram-бот для мониторинга штрафов Wildberries в реальном времени.

**Проблема:** Wildberries списывает штрафы без уведомления. Селлеры узнают о них слишком поздно — когда дедлайн обжалования (7 дней) уже прошёл. Новички теряют 50-150K руб. на штрафах.

**Решение:** Бот проверяет финансовые отчёты WB каждые 30 минут и мгновенно уведомляет о новых штрафах с суммой, причиной, номером поставки и дедлайном обжалования.

## Возможности

- Мгновенные алерты о новых штрафах в Telegram
- Сумма, причина, артикул, поставка, дедлайн обжалования
- История штрафов и статистика (неделя/месяц/всего)
- Мульти-магазин (несколько API-ключей)
- Freemium: 1 магазин бесплатно, PRO до 20 магазинов

## Стек

- **aiogram 3** — Telegram Bot API
- **aiohttp** — HTTP-клиент для WB API
- **aiosqlite** — асинхронный SQLite
- **APScheduler** — периодическая проверка штрафов

## Установка

```bash
git clone https://github.com/your-username/wb-shtraf-shield.git
cd wb-shtraf-shield
pip install -r requirements.txt
cp .env.example .env
# Заполните .env: BOT_TOKEN и ADMIN_IDS
```

## Запуск

```bash
python bot.py
```

## Как получить API-ключ WB

1. Зайдите на [seller.wildberries.ru](https://seller.wildberries.ru)
2. Настройки → Доступ к API
3. Создайте новый ключ с типом **«Статистика»**
4. Скопируйте ключ и вставьте в бота

> Ключ «Статистика» даёт только чтение данных. Изменить товары или цены через него невозможно.

## Как это работает

1. Селлер добавляет API-ключ WB через бота
2. Каждые 30 минут бот запрашивает финансовый отчёт через WB API
3. Из отчёта извлекаются операции со штрафами (`penalty > 0`)
4. Новые штрафы (не виденные ранее) — мгновенный алерт в Telegram
5. Рассчитывается дедлайн обжалования (7 дней от даты штрафа)

## Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `BOT_TOKEN` | Telegram Bot Token | — |
| `ADMIN_IDS` | ID администраторов (через запятую) | — |
| `CHECK_INTERVAL` | Интервал проверки (минуты) | 30 |
| `REPORT_LOOKBACK_DAYS` | Глубина проверки (дни) | 7 |
